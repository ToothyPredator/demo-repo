Page &
Replacement
The page replacement algorithm decides which memory page is to be replaced. The process of replacement is sometimes called swap out or write to disk. Page replacement is done when the requested page is not found in the main memory (page fault). There are two main aspects of virtual memory, Frame allocation and Page Replacement.
